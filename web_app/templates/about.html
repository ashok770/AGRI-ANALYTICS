Project Methodology & Goal

This application provides an accurate and transparent forecast of crop yields to help farmers and agribusinesses make better resource decisions.

The Problem: The “Black Box” Gap

Traditional machine learning models predict outcomes but fail to explain why they reached that conclusion. In agriculture, a farmer needs to know whether low yield is caused by fertilizer usage, rainfall, or temperature to take corrective action.

Our goal was to solve this Explainability Gap by building an analytical application that delivers both predictions and root-cause insights.

The Solution: An Explainable ML Pipeline

1. Data Sources

The model was trained on historical global datasets including Rainfall, Average Temperature, Pesticide Usage, Crop Type, and Geographic Area.

2. Core Model

We used a Random Forest Regressor, an ensemble model well-suited for capturing complex, non-linear relationships in agricultural data.

3. Performance

The model achieved an R² score of ~0.9875, explaining over 98% of yield variability — indicating strong predictive reliability.

Key Analytical Insight (The “WHY”)

Feature importance analysis revealed the top global drivers of yield:

  • #1: Crop Type (e.g., Potatoes)
  • #2: Pesticide Tonnes Applied
  • #3: Average Temperature

By highlighting these factors, the application converts predictions into actionable decision support.
